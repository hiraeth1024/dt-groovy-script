



JSON数据爬取所需字段

| pageTitle         | 路径         | 以`\`分割，从首页开始一步一步到目标爬取页面 |      |
| ----------------- | ------------ | ------------------------------------------- | ---- |
| 字段名            | 含义         | 取值逻辑                                    | 备注 |
| data_target_table | 数据目标表   |                                             |      |
| release_frequency | 披露频率     | 数据以什么间隔发布                          |      |
| end_date          | 截止日期     | 根据披露频率取最后一日                      |      |
| indicator_name    | 指标名称     | 以`\`分割，一般是表格的标题+指标数据的含义  |      |
| indicator_data    | 指标数据     |                                             |      |
| indicator_unit    | 指标单位     |                                             |      |
| title             | 标题         |                                             |      |
| pushtime          | 信息发布时间 |                                             |      |
| raw_info_date     | 原始发布日期 |                                             | 可选 |
| raw_end_date      | 原始截止日期 |                                             | 可选 |
| fileType          | 文件类型     | 默认取值`json`                              |      |
| statistic_area    | 统计区域     |                                             |      |
| title             | 标题         | 标题*所属表                                 |      |



文件类型数据爬取所需字段

| 字段名            | 含义         | 取值逻辑                                                     | 备注 |

| pageTitle         | 路径         | 以`\`分割，从首页开始一步一步到目标爬取页面                  |      |
| raw_info_date     | 原始发布日期 |                                                              | 可选 |
| pushtime          | 信息发布时间 |                                                              |      |
| downloadUrl       | 文件下载链接 |                                                              |      |
| fileMd5           | 文件MD5      |                                                              |      |
| title             | 标题         | 标题*所属表, 根据数据目标表对应`*国际宏观，*新宏观，*新宏观（海关贸易），*新宏观（行业财务），*新宏观（区域），*行业，*地方经济` |      |
| filename          | 文件名       |                                                              |      |
| fileSize          | 文件大小     |                                                              |      |
| webUrl            | 网站地址     |                                                              |      |
| data_target_table | 数据目标表   | usrGJHGSJB，usrXHGSJBHG，usrXHGSJBHGMY，usrXHGSJBHGCW，usrXHGSJBQY，usrHYSJB，usrDFJJYXSJB |      |
| detailUrl         | url链接      |                                                              |      |
| fileType          | 文件类型     |                                                              |      |
